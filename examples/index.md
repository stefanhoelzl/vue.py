## Run Examples Local

```bash
$ git clone https://github.com/stefanhoelzl/vue.py.git
$ cd vue.py
$ make env.up
$ cd examples/<example-name>
$ python vuecli/cli.py deploy flask
```
Goto [http://localhost:5000](http://localhost:5000)
